# 项目约定

## 术语约定：参考竞品项目

当周浩提到"参考竞品项目""竞品参考"时，**默认指**以下目录：

`projects/competitor-references/01-desktop-apps/`

该目录下现有 8 个桌面端 Agent 竞品：MyAgents、hexclaw-desktop、kuse-cowork、harnessclaw、lobsterai、Kun、opencowork、AionUi。

- 需要找竞品实现做对比、借鉴设计、查证某个功能怎么做时，先到这个目录下找。
- 如需参考其他形态的竞品（命令行类 02/03、框架类 04、自托管 Web 类 05、多端引擎类 06），周浩会另行指明具体分类目录。
