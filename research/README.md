# research/ — 竞品知识库

26 款 AI Agent 产品的资料库。模仿 Karpathy LLM Wiki 模式组织。这是管道的产出物存放地，也是文章系列的素材来源。

| 子目录 | 装什么 |
|---|---|
| `entities/` | 每款产品的档案 |
| `compiled/` | 编译后的产品评测卡 |
| `raw/` | 原始素材：论文、官方文档、社区文章 |
| `topics/` | 主题研究 |
| `concepts/` | 概念词条 |
| `comparisons/` | 产品横向对比 |
| `changelog/` | 每日变更记录 |
| `reports/` | 生成的报告（daily/weekly/on-demand/portfolio） |
| `lessons/` | 踩坑经验 |
| `timelines/` | 设计演进时间线 |
| `review/` | 素材审核队列 |
| `_framework/` | 知识库组织规范 |
| `schema/` | 数据结构定义 |
